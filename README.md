<b>PUMP IT UP RANDOMIZER</b>
</div>   

This is a project made for a local Pump it up community and for personal uses. The project fetches API and then it generates a random chart where you can choose from singles and doubles. It solves the problem of not knowing which chart or difficulty to play for that specific session.

API Documentation I used: https://piuscores.arroweclip.se/swagger/index.html
Work in progress.
